Science Toolbox

This project is built using c++ and graphics.h 

#include <iostream> 
#include <graphics.h>
#include <math.h>
.....

(✨ i know its old but still ✨). This project includes a Light Simulator, Linear Grapher and an Organic Compound Namer.

Compound Namer:

Can name straight chain compounds from 1-10 c-atoms and various substituent groups.Completly integrated in graphics window.

Light Simulator:

Simulates propagation of light in medium along with reflection of light against convex and concave mirror with lines indicating light. Output in both command line and graphics window.

Linear Grapher:

Graphs linear equations of form y=mx+c. Output in both command line and graphics window.
